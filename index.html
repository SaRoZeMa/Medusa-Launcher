O Projeto

A Medusa é uma plataforma criada para reunir e divulgar jogos independentes desenvolvidos pela nossa equipe. O objetivo é oferecer um espaço centralizado onde qualquer pessoa possa descobrir, jogar e acompanhar os projetos que criamos com dedicação — tudo em um único lugar, com uma interface limpa e moderna.

Este projeto nasceu como trabalho escolar. Cada membro contribui com suas habilidades únicas para que a plataforma funcione de ponta a ponta.
